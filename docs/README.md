# Project Documentation

This folder contains supporting documentation for the Restaurant Ordering and Management System final-year project.

## Purpose

The documentation folder is included to make the GitHub repository more complete and easier to assess. It supports the final report by showing evidence of planning, testing, project management, and viva/demo preparation.

## Recommended Files

The following evidence can be added to this folder before final submission:

```text
docs/
  README.md
  gantt-chart.md
  viva-demo-checklist.md

```

## How This Supports the Project

- The Gantt chart supports project planning and progress tracking.
- The Postman collection supports backend API verification.
- The testing report supports validation and evaluation.
- The diagrams support requirements analysis and system design.
- The viva checklist supports final demonstration preparation.

The main report explains the full project lifecycle. This folder stores the practical supporting evidence.
